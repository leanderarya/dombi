Ringkasan Perubahan

Dashboard

  • KPI 2×2 card grid → 4-wide horizontal strip (bg-only, no borders)
  • Action list: border-card items → bg-only soft list
  • Stok Kritis: separate card section → same bg-list style
  • Hero bar: tighter padding, smaller text

Orders & List Pages

  • Card list → flat table rows w/ column headers
  • Sticky sidebar stats → KPI strip di atas table
  • No card shadows, hanya border separators
  • Hover: light bg color shift, no lift
